# Telegram Shop Bot

Telegram-бот магазина на Python и aiogram 3.

Проект сделан как учебный и портфолио-проект для практики backend-архитектуры Telegram-ботов.

## Возможности пользователя

* Просмотр каталога товаров
* Просмотр карточки товара с описанием, ценой, остатком и фото
* Покупка товара с баланса
* Просмотр баланса
* История покупок с датой
* Профиль покупателя
* Просмотр популярных товаров

## Возможности администратора

* Добавление товара
* Добавление описания и фото товара
* Архивация товара вместо удаления
* Восстановление товара из архива
* Пополнение остатка товара
* Изменение цены товара
* Изменение названия товара
* Изменение описания товара
* Изменение или удаление фото товара
* Пополнение баланса пользователя
* Просмотр пользователей
* Просмотр финансовых операций пользователя
* Статистика магазина
* Админская справка по командам

## Архитектура проекта

Проект построен по схеме:

```text
Handler → Service → Database
```

Handlers принимают сообщения и callback-запросы от пользователя.

Services содержат бизнес-логику: проверки, расчёты, формирование ответов.

Database отвечает за работу с SQLite: товары, пользователи, покупки, баланс, транзакции.

## Безопасность

Токен бота и ID администратора хранятся в `.env`.

Файл `.env` не загружается в GitHub.

Для примера настроек используется файл `.env.example`.

## Установка

1. Клонировать репозиторий:

```bash
git clone https://github.com/epifanovamur-cyber/telegram-shop-bot.git
```

1. Перейти в папку проекта:

```bash
cd telegram-shop-bot
```

1. Создать виртуальное окружение:

```bash
python -m venv .venv
```

1. Активировать виртуальное окружение:

```bash
.venv\Scripts\activate
```

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

1. Создать файл `.env` по примеру `.env.example`:

```env
BOT_TOKEN=your_bot_token_here
ADMIN_ID=your_telegram_id_here
```

1. Запустить бота:

```bash
python bot.py
```
